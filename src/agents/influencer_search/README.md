# Influencer Discovery Agent

This agent helps small businesses find suitable Instagram creators to contact for
collaborations.

## What it does
- Takes a high-level business description from the user
- Uses the Firecrawl MCP server to discover relevant creators and public information
- Analyzes creator bios, content themes, tone, and collaboration signals
- Recommends a short, curated list of creators to contact
- Explains why each creator is a good fit

## What it does not do
- It does not rely on follower counts or engagement metrics
- It does not scrape Instagram directly
- It does not automate outreach or messaging
